My Portfolio

Portfolio Website

This multi-page portfolio website was designed from scratch using Figma wireframes and then developed with HTML, CSS, and JavaScript. The site includes a responsive layout, custom navigation system, and individually designed project pages. My goal was to create a clean and modern front-end experience that highlights my development work, creativity, and technical skills.

Features

  • Responsive multi-page layout
  • Sidebar navigation with active state styling
  • Projects with individual detailed pages
  • Clean HTML and CSS structure
  • Designed fully in Figma before development
  • Deployed using Netlify

Tech Used

HTML CSS JavaScript Figma Netlify
